Uamuzi Bora

Uamuzi Bora is a project to create an electronic medical record system for HIV-clinics in Kenya.
The pilot database is installed at the Provicial General Hospital in Kakamgea in Western Kenya.


Uamuzibora Labs

Uamuzibora Labs is a web based application to display statistics generated from the data collected through
the uamuzibora electronic medical record system for HIV patients. All data is completly anomized.
